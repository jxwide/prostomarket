* ProstoMarket - российский онлайн маркет

# Get start

## Installation
client (nextJS)
```bash 
    cd client
    npm i
    cd ..
```
server (nestJS)
```bash 
    cd server
    npm i
    cd ..
```

## Running
client (nextJS)
```bash 
    cd client
    npm run dev
```
server (nestJS)
```bash 
    cd server
    npm run start:dev
```
